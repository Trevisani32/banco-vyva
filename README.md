# Banco VYVA

Projeto de sistema bancário em Java.

## Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/Trevisani32/banco-vyva.git`
2. Abra no VS Code
3. Execute o `Main.java`

## Estrutura do projeto
- `model/` — classes de dados
- `service/` — regras de negócio
- `repository/` — armazenamento em memória
- `exception/` — erros customizados

## Regras de contribuição
- Nunca commite direto na `main`
- Crie uma branch com o padrão: `feature/nome-da-tarefa`
- Abra um Pull Request e aguarde a revisão do Igor
